# 홍서당 (紅緖堂) — 정밀 사주 감정 + 紅線 인연 매칭

붉은 실마리의 집. 자미두수 명반으로 인생 12개 영역과 10년 대운을 감정하고, 紅線으로 인연을 잇는다.

퍼널: 허브(등불 선택) → 롱폼 랜딩(/reading) → 문답 6문 → 무료 진단(명궁 공개+봉인 4장) → 3단 상품 결제(토스 딥링크/무통장) → 감정서 문자 전달 → 紅線 매칭 → 재방문(내 감정서/인연함).

- 스택: Next.js 14 · Supabase · Vercel(크론 2개) · Solapi · Claude API
- 운영: /admin 콕핏 — 결제확인 체크 1번으로 리포트 생성·문자까지 자동
- 상세: 프로젝트 인수인계 문서 참조
